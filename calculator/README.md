# 计算器程序 - Agent Engineer

由 Agent Engineer 开发的现代化计算器应用，包含基础计算器和科学计算器两个版本。

## 🚀 快速开始

### 本地运行
```bash
# 启动本地服务器
cd calculator
python3 -m http.server 8080

# 访问计算器
# 基础计算器: http://localhost:8080/index.html
# 科学计算器: http://localhost:8080/scientific.html
# 演示页面: http://localhost:8080/demo.html
```

## 📖 使用指南

### 基础计算器操作

#### 基本运算
1. 输入数字: 点击数字按钮或使用键盘
2. 选择运算符: 点击 +, -, ×, ÷ 按钮
3. 计算结果: 点击 = 按钮或按 Enter 键

#### 高级功能
- **清除 (C)**: 清除当前输入
- **删除 (←)**: 删除最后一个字符
- **正负号 (±)**: 切换正负号
- **百分比 (%)**: 计算百分比
- **平方根 (√)**: 计算平方根
- **平方 (x²)**: 计算平方
- **倒数 (1/x)**: 计算倒数

#### 内存功能
- **MC**: 清除内存
- **MR**: 调用内存值
- **M+**: 将当前值加到内存
- **M-**: 从内存减去当前值

### 科学计算器操作

#### 三角函数
- sin, cos, tan: 计算三角函数值
- asin, acos, atan: 计算反三角函数值

#### 对数和指数
- log: 常用对数 (以10为底)
- ln: 自然对数 (以e为底)
- eˣ: 指数函数
- xʸ: 幂运算

#### 数学常数
- π: 圆周率 (3.14159...)
- e: 自然常数 (2.71828...)

#### 其他功能
- **n!**: 阶乘计算
- **|x|**: 绝对值
- **⌊x⌋**: 向下取整
- **⌈x⌉**: 向上取整

### 键盘快捷键

#### 基础计算器
- `0-9`: 数字输入
- `+`, `-`, `*`, `/`: 运算符
- `Enter`, `=`: 计算结果
- `Escape`, `C`: 清除
- `Backspace`: 删除

#### 科学计算器
- 所有基础计算器快捷键
- `(`, `)`: 括号
- `p`: π 常数
- `e`: e 常数
